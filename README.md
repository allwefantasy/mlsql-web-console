# mlsql-web-console

mlsql-web-console是一个基于React的前端应用,
是为了 [MLSQL](https://github.com/allwefantasy/mlsql)配套开发的一套Web UI系统。

提供的功能有：

1. 用户权限体系
2. 支持类似notebook的交互模式
3. 支持普通的MLSQL脚本
4. 支持脚本管理

对应的后端项目在：https://github.com/allwefantasy/mlsql-api-console

界面操作演示：https://www.bilibili.com/video/av83093613
